Flexnet TinyMCE 5 SOME plugin

A plugin for adding media content to TinyMCE 5.
Add Facebook, Twitter, YouTube, Vimeo, Soundcloud, Infogram

Requires jQuery and PHP7+.

  • Place the cursor in the textarea where you want the media embedded.
  • Click on the plugin button
  • Insert the URL for the embed and press OK.
    E.g.:
    https://vimeo.com/8870338 or
    https://www.facebook.com/uniladmag/videos/2530516633638047/
    https://youtu.be/tUEJkwA1VI4
You can find the full code on GitHub here:
https://github.com/wegeberg/flexnet-tinymce5-some
